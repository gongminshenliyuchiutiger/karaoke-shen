# 手機點歌連線疑難排解 (Mobile Connectivity Help)

如果您掃描 QR Code 後手機顯示「無法連線」，請檢查以下步驟：

### 1. 確保在同一個 Wi-Fi 網路
- 電腦與手機必須連接到同一個路由器或 Wi-Fi 分享器。
- 請檢查手機是否誤開了行動數據（4G/5G）而沒連上 Wi-Fi。

### 2. Windows 防火牆設定 (重要!)
Windows 可能會自動封鎖外部連線，請嘗試以下操作：
1. 開啟「控制台」 > 「系統及安全性」 > 「Windows Defender 防火牆」。
2. 點選左側的「允許應用程式或功能通過 Windows Defender 防火牆」。
3. 點選「變更設定」。
4. 找到 `karaoke-shen.exe` (或您的 Python 程式)，確保 **「私人」** 與 **「公用」** 都有勾選。
5. 如果找不到，點選「允許其他應用程式」並手動加入執行檔。

### 3. 防毒軟體
- 部份防毒軟體（如 Avast, Norton, 卡巴斯基）有獨立的防火牆，請暫時關閉或將程式加入白名單。

### 4. 網路設定為「私人」
1. 電腦點擊右下角 Wi-Fi 圖示 > 「屬性」。
2. 將「網路設定檔」從「公用」改為 **「私人」**。這會讓電腦在區域網路中可被看見。

---
完成上述步驟後，請**重新啟動**程式並再次嘗試掃描。
